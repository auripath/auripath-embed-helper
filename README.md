# Auripath Embed Helper

Auripath Embed Helper is the official WordPress plugin for adding hosted Auripath audio experiences to WordPress pages and posts.

Auripath adds audio, lead capture and CTA tracking to WordPress so B2B teams can turn PDF traffic into sales signals.

## What it does

The plugin adds a simple shortcode:

[auripath doc="doc_xxxxxxxxx"]

Replace `doc_xxxxxxxxx` with the public document ID from your Auripath account.

The plugin loads the hosted Auripath embed script, renders the selected audio experience, and lets Auripath handle the player, lead capture, CTAs, listening analytics and conversion tracking.

## Use it for

- Adding audio to B2B PDFs, guides, reports and sales assets
- Capturing leads around existing content
- Tracking who listened and who clicked
- Giving sales a clearer reason to follow up
- Adding a read-or-listen experience to WordPress landing pages

## External service

This plugin connects to the hosted Auripath service at:

https://app.auripath.com/

When a page containing the Auripath shortcode is viewed, the plugin loads the Auripath embed script from the hosted Auripath service. The public document ID in the shortcode is passed to Auripath so the hosted service can return the correct player, lead capture form, calls to action and analytics behavior for that document.

The plugin itself does not store leads, audio files or analytics data in WordPress.
